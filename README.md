# Python Learning Repository 🐍

Welcome to this Python learning journey! This repository contains beginner-level Python scripts covering fundamental concepts and operations.

## 📚 Repository Overview

This is a learning-focused repository designed to help beginners understand core Python concepts from scratch. All files are written in Python and demonstrate basic programming principles through practical examples.

## 📂 Files and Contents

### Demo Files
- **demo.py** - Introduction script
  - Simple print statement demonstrating basic output
  - "Hello World" style introduction

- **demo day 2.py** - Arithmetic Operations
  - Demonstrates all basic arithmetic operators: `+`, `-`, `*`, `/`, `//`, `%`, `**`
  - Variable swapping using tuple unpacking
  - Operations on numbers: `a=10, b=3`

### Input and String Manipulation

- **3.py** - Basic Input Handling
  - Simple program to accept user input and display it
  - Introduction to the `input()` function

- **3.1.py** - Relationship Age Difference Calculator
  - Accepts input for two people's names and ages
  - Calculates age difference using `abs()` function
  - Demonstrates multiple string concatenation methods:
    - Traditional string concatenation
    - f-string formatting (modern approach)

- **3.2.py** - Personalized Greeting
  - Takes user's name and age as input
  - Displays personalized greeting messages
  - Shows both string concatenation and f-string formatting

- **3.3.py** - String Manipulation Operations
  - Demonstrates various string methods:
    - `upper()` - Convert to uppercase
    - `lower()` - Convert to lowercase
    - `strip()` - Remove excess whitespace
    - `replace()` - Replace text within strings
    - `len()` - Get string length
  - Interactive input-based examples

## 🎯 Learning Concepts Covered

### Core Programming Fundamentals
1. **Variables and Data Types**
   - Integer and string types
   - Variable assignment and manipulation

2. **Input and Output**
   - `input()` function for user input
   - `print()` function for output

3. **Arithmetic Operations**
   - Basic operators: `+`, `-`, `*`, `/`
   - Integer division: `//`
   - Modulus: `%`
   - Exponentiation: `**`

4. **String Operations**
   - String concatenation
   - F-string formatting
   - String methods: `upper()`, `lower()`, `strip()`, `replace()`
   - String length with `len()`

5. **Type Conversion**
   - Converting strings to integers with `int()`
   - Converting integers to strings with `str()`

6. **Advanced Concepts**
   - Tuple unpacking for variable swapping
   - Absolute value calculation with `abs()`

## 🚀 How to Use

1. **Clone the repository**
   ```bash
   git clone https://github.com/santhosh-hd-ctrl/python.git
   cd python
   ```

2. **Run any script**
   ```bash
   python3 filename.py
   ```

3. **Example: Running demo.py**
   ```bash
   python3 demo.py
   ```

4. **Interactive scripts with input**
   - For scripts like `3.1.py`, follow the prompts:
   ```bash
   python3 3.1.py
   # Enter boy name: John
   # Enter boy age: 25
   # Enter girl name: Jane
   # Enter girl age: 23
   ```

## 💡 Learning Path Suggestion

Start with the files in this order:

1. `demo.py` - Basic print statement
2. `3.py` - Simple input/output
3. `3.2.py` - Working with multiple inputs
4. `demo day 2.py` - Arithmetic operations
5. `3.1.py` - Combining input, arithmetic, and strings
6. `3.3.py` - Advanced string operations
